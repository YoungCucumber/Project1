## Пояснительная записка к проекту "EGEstresses"
### Анализ и актуальность проекта
В целях облегчения подготовки выпускников к успешному выполнению 4 задания ЕГЭ по русскому языку перед нами\
ставится задача разработки функционального приложения, которое будет обучать выпускников словами из открытого\
банка ЕГЭ, в которых нужно верно поставить ударения, в интерактивной форме, что будет способствовать легкому\
запоминанию материала.
### Цель проекта
Разработка приложения-тренажёра " EGEstresses " для подготовки к ЕГЭ по русскому языку на языке\
программирования Python с использованием библиотеки PyQT5.
####Таблицы базы данных

Таблица Users хранит в себе информацию о кадом пользователе. 
1) Id
2) Логин
3) Пароль
4) Избранные пользователем слова. Здесь подключен внешний ключ к таблице Favourites

#### 1. Вход

<img width="797" alt="image" src="https://user-images.githubusercontent.com/92947429/140497298-281a9243-b1f7-48eb-b2ae-0ea8d10ded65.png">


__Краткое описание:__ При входе в приложение пользователю будет предложено авторизоваться. Можно создать новый аккаунт, либо войти в свой\
аккаунт. Информ
__Функционал:__
- При входе пользователю необходимо ввести логин и пароль от своего аккаунта в LineEdit(-ы). Все данные храняться в таблице базы данных. При некорректном вводе вызывается ошибка и выводится в StatusBar (в случпе, если одно или оба поля для ввода пустые, либо если такого аккаунта не существует, либо пароль не соответствует логину). Вся информация импортируется из таблицы базы данных Users
- Если у пользователя нет акккаунта, он может создать новый, нажав на кнопку "Регистриция". После нажатия на эту кнопку данное окно закрывается и открывается новое для регистрации
- Если все хорошо, то окно закрывается и открывается окно "Меню"
#### 2. Регистрация

<img width="790" alt="image" src="https://user-images.githubusercontent.com/92947429/140497349-2a6ff2bf-c160-446b-977d-e16c6a1660a7.png">

__Краткое описание:__ Пользователь регестрируется в приложении и новая информация о нем сохраняется в таблицу базы данных Users.
__Функцонал:__
- При нажатии на кнопку "ОК" идет проверка на логин и пароль, описанная в следующих пунктах
- Если при вводе логина оказывается, что пользователь с таким именем уже существует, то программа сообщает об ошибке и печатает ее в StatusBar
- Если все хорошо, то программа закрывает окно и открывает окно "Меню"
- Информация добавляется в таблицу базы данных Users

- При нажатии на кнопку "Карточки" откроется окно "Карточки"
- При нажатии на кнопку "Тест" откроется окно "Тест"
#### 4. Все слова

<img width="792" alt="image" src="https://user-images.githubusercontent.com/92947429/140498473-4b71a456-31e4-400b-9bf7-c54ea29c7e99.png">

__Краткое описание:__ Перед пользователем будет дан список всех слов из таблицы базы данных list_of_words.
__Функционал:__
- При нажатии на кнопку "Назад" окно "Все слова" закроется и откроется окно "Меню"
- В Scroll Bar лежит FormLayout с check box, на которых написаны все слова из базы данных. По желание, человек может поставить галочку
и добавить слово в избранное, или наоборот. При выходе из данного окна слова сохранятся в таблицу базы данных Favourites с Id, который соответствует пользователю, который в данный момент использует приложение
-При нажатии на кнопку "Инструкция" на экране появится MessageBox с инструкцией, как добавлять или убирать слова из избранных

- При нажатии на кнопку "Назад" окно "Избранное" закроется и откроется окно "Меню"

#### 6. Заучивание избранных слов

<img width="794" alt="image" src="https://user-images.githubusercontent.com/92947429/140498775-86139e5d-9f5c-4b14-bcec-25920ccdb698.png">

__Краткое описание:__ На большой кнопке-карточке будут по очереди отображаться избранные человеком слова строчными буквами. Если пользователь захочет посмотреть правильный ответ, то ему нужно лишь нажать на эту кнопку. Если польхователь посчитал, что он выучил это слово, то может нажать на кнопку "Усвоено" и слово больше ему не встретится. Также для переключения слова нужно нажать на кнопку "Следующее".
__Функционал:__
- При нажатии на кнопку "Назад" окно "Заучивание избранных слов" закроется и откроется окно "Избранное"
- При нажатии на кнопку она поменяет свой цвет на красный и на ней отобразится верный ответ. Также у пользователя будет возможность обратно перевернуть "карточку"
- При нажатии на кнопку "Усвоено" слово удалится из временного списка и больше это слово не будет встречаться при дальнейшем просматривании "карточек"
- При нажатии на кнопку "Следующее" перед пользователем появится следующее слово
- В ProgressBar в процентах будет показываться, сколько вопросов осталось до конца "карточек", пока пользователь не отметит все слова, как "усвоенные"
#### 7. Карточки

<img width="797" alt="image" src="https://user-images.githubusercontent.com/92947429/140499120-1c1319e0-3ca5-4852-8561-25a0f767d174.png">

__Краткое описание:__ Перед началом заучивание пользователю необходимо выбрать количество слов, которое он хочет выучить. После этого на большой кнопке-карточке будут по очереди отображаться слова строчными буквами. Если пользователь захочет посмотреть правильный ответ, то ему нужно лишь нажать на эту кнопку. Если польхователь посчитал, что он выучил это слово, то может нажать на кнопку "Усвоено" и слово больше ему не встретится. Также для переключения слова нужно нажать на кнопку "Следующее". При желании пользователь может добавить слово в "Избранное", поставив галочку в check box. При желании, может убрать галочку. 
__Функционал:__
- При нажатии на кнопку "Назад" окно "Карточки" закроется и откроется окно "Меню"
- В SpinBox пользователь выбирает количество слов и нажимает на кнопку "Применить"
- На кнопке будут в рандомном порядке отображаться слова . Слова будут напечатаны в нижнем регистре и с заглавной буквы
- При нажатии на кнопку она поменяет свой цвет на красный и на ней отобразится верный ответ. Также у пользователя будет возможность обратно перевернуть "карточку"
- При нажатии на кнопку "Усвоено" слово удалится из временного списка и больше это слово не будет встречаться при дальнейшем просматривании "карточек"
- При нажатии на кнопку "Следующее" перед пользователем появится следующее слово
- При установке галочке на CheckBox слово добавляется в "избранное". Если слово уже было предварительно добавлено в избранное,
то check box будет отмечен галочкой по дефолту
- В ProgressBar и на SpinBox будет показываться, сколько вопросов осталось до конца "карточек", пока пользователь не отметит все слова, как "усвоенные"
#### 8. Тест

<img width="795" alt="image" src="https://user-images.githubusercontent.com/92947429/140500718-f047f034-0ba4-4fdc-87b6-1b8957b86ced.png">

__Краткое описание:__ Перед началом пользователь выбирает количество вопросов. Далее перед ним будут по очереди появляться слова строчными буквами и варианты ответов на кнопках. После выбора правильного ответа появляется новое слово. Когда вопросы кончатся, то покажутся результаты.
__Функционал:__
- При нажатии на кнопку "Назад" окно "Карточки" закроется и откроется окно "Меню"
- В SpinBox пользователь выбирает количество вопросов и нажимает на кнопку "Применить"
- Программа будет высчитывать количество гласных и взависимости от этого регурировать количество кнопок, добавленных в VerticalLayout
- В Label будут по очереди отобраться слова из таблицы базы данных. Слова будут в нижнем регистре и с маленькой буквы
- При выборе ответа будет сразу открываться следующее слово
- В ProgressBar и на SpinBox будет показываться, сколько вопросов осталось до конца теста
- Когда слова закончатся, то окно "Тест" закроется и откроется окно "Результаты"
